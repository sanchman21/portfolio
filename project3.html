Predicting Heart Disease using ML

Diagnosing a person with heart disease can sometimes be a time taking process. The aim of this project was to predict whether the person has heart disease or not using certain medical parameters. Data was accessed using Kaggle which has 12 features and 1 target column.

This project is an end-to-end Kaggle ML notebook where I first cleaned the data, then performed EDA on it, handled outliers, did data preprocessing for numerical and categorical features and then performed Machine Learning and Deep Learning along with Cross Validation and Hyperparameter Optimization. I have received Kaggle Bronze Medal for this project.
